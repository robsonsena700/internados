# Changelog - Painel de Monitoramento de Pacientes

Todas as alterações notáveis neste projeto serão documentadas neste arquivo.

## [1.0.0] - 2026-02-12
### Adicionado
- Versão inicial do Painel de Monitoramento.
- Dashboard responsivo com indicadores em tempo real.
- Filtros avançados por médico, paciente, unidade e especialidade.
- Sistema de autenticação seguro com Bcrypt e Sessões.
- Infraestrutura Docker para deployment simplificado.
- Auditoria de segurança completa (OWASP Top 10).
- Rodapé dinâmico com versionamento automático.
- Scripts de deployment automatizado para o servidor.
