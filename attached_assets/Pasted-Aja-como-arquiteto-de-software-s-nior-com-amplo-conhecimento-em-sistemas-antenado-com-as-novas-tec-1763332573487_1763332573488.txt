A primeira consulta de vamos criar é a de pacientes internados. Pra isso, utilize todos os pacientes em que o valor de sotech.ate_atendimento.fktipoatendimento for igual a 2 e o valor de datasaida for null.

Traga os seguintes filtros:

Médico assistente: fkprofissionalatendimento
Paciente: fkpaciente
Unidade de atendimento: fkunidadesaude
Leito: fkleito
Data de internação: dataentrada
Especialidade: fkespecialidade
Procedimento solicitado: fkprocedimentosolicitado

Todos os dados que iniciam do fk são chaves estrangeiras, devendo ser apresentados nos filtros os valores destas chaves obtidos a partir das tabelas relacionadas. Para isso, no mapeamento dos dados, busque toda integridade referencial necessária.

Nesta primeira implementação, traga na lista os mesmos dados que foram informados para os filtros. Depois vamos ajustando.

O banco de dados será configurado via secrets, por tanto, para realização do mameamento do banco de dados, será necessário que os variáveis de conexão sejam informadas.
